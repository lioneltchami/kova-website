# kova report

Generate a detailed cost report for a period. Reports pull from the cloud dashboard so they include records from all your machines. Use `--format csv` or `--format json` to export data for spreadsheets or billing systems.

## Usage

```bash
kova report [flags]
```

## Flags

| Flag        | Type   | Default      | Description                                                                                |
| ----------- | ------ | ------------ | ------------------------------------------------------------------------------------------ |
| `--period`  | string | `this-month` | Reporting period: `this-month`, `last-month`, `this-quarter`, `last-quarter`, or `YYYY-MM` |
| `--format`  | string | `table`      | Output format: `table`, `csv`, `json`                                                      |
| `--output`  | string | stdout       | Write output to this file path instead of the terminal                                     |
| `--by`      | string | `tool`       | Group by: `tool`, `model`, `project`, `member`, `day`                                      |
| `--tool`    | string | all          | Filter to a specific tool                                                                  |
| `--project` | string | all          | Filter to a specific project                                                               |

### Export Last Month as CSV

```bash
kova report --period last-month --format csv --output march-2026-costs.csv
```

The CSV includes one row per usage event with columns: `date`, `tool`, `model`, `project`, `input_tokens`, `output_tokens`, `cost_usd`.

### Export as JSON (for Billing Integration)

```bash
kova report --period this-month --format json --output costs.json
```

```bash

```

## Authentication

Reports pull from the cloud API. The API key from `kova login` is used automatically. Ensure you have synced recent records with `kova sync` before generating a report.
