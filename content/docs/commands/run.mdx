# kova run

Execute an AI coding task with intelligent model routing. Kova analyzes prompt complexity, selects the appropriate model tier, loads file context, streams the response, and records the cost. Supports smart model fallback on rate limits and session budget guards.

## Usage

```bash
kova run <prompt> [flags]
```

## Flags

| Flag           | Type    | Default | Description                                                             |
| -------------- | ------- | ------- | ----------------------------------------------------------------------- |
| `--model`      | string  | auto    | Use a specific model (e.g. `anthropic:claude-sonnet-4-20250514`)        |
| `--provider`   | string  | auto    | Use a specific provider (`anthropic`, `openai`, `google`, `openrouter`) |
| `--tier`       | string  | auto    | Use a model tier: `cheap`, `mid`, or `strong`                           |
| `--dry-run`    | boolean | false   | Show what would happen without making changes                           |
| `--auto-apply` | boolean | false   | Apply file edits without confirmation                                   |
| `--context`    | string  | -       | Attach a file as context (repeatable)                                   |
| `--include`    | string  | -       | Include files matching a glob pattern as context (repeatable)           |
| `--budget`     | string  | -       | Session budget cap in USD (e.g. `1.50`)                                 |

### Attach additional context files

```bash
kova run "refactor the API handler to match the new schema" \
  --context src/schemas/user.ts \
  --include "src/handlers/**/*.ts"
```

### Dry run to preview model selection

```bash
kova run "rewrite the auth module" --dry-run
```

```
  [DRY RUN] Would execute with:
    Complexity: complex
    Model: claude-opus-4-20250514 (anthropic)
    Tier: strong
    Context files: 5
    Budget: none set
```

### Set a budget cap

```bash
kova run "optimize the database queries" --budget 0.50
```

If the session cost approaches the budget cap, Kova will warn you and stop before exceeding it.
