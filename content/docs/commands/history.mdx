# kova history

View past AI session history with costs. Groups usage records by session, showing date, cost, model, project, token count, duration, and number of turns. Sessions are sorted by most recent first.

## Usage

```bash
kova history [flags]
```

## Flags

| Flag        | Type   | Default | Description                                              |
| ----------- | ------ | ------- | -------------------------------------------------------- |
| `--tool`    | string | all     | Filter by tool (e.g. `kova_orchestrator`, `claude_code`) |
| `--project` | string | all     | Filter by project name                                   |
| `--days`    | number | `30`    | Look back N days                                         |
| `--limit`   | number | `50`    | Maximum number of sessions to show                       |

```bash
kova history --tool claude_code
```


### Filter by project

```bash
kova history --project myapp --days 7
```

### Increase the lookback window

```bash
kova history --days 90 --limit 100
```
