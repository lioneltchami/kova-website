# Installation

Kova is an npm CLI that tracks cost and usage across 11 AI coding tools: Claude Code, Cursor, GitHub Copilot, Windsurf, Devin, Aider, Continue.dev, Cline, Amazon Q, Bolt, and Lovable. Install it once and get a unified view of what your AI tools cost.

## Prerequisites

| Requirement | Minimum Version | Check            |
| ----------- | --------------- | ---------------- |
| **Node.js** | 18+             | `node --version` |
| **Git**     | Any             | `git --version`  |

You do not need Claude Code or any other AI tool pre-installed to track costs -- Kova collects usage data from whichever tools you already use.

## Install Methods

### Global Install (Recommended)

Install Kova globally so the `kova` command is available in every project:

```bash
npm install -g kova-cli
```

Verify the installation:

```bash
kova --version
```

### Use Without Installing (npx)

Run Kova without a global install:

```bash
npx kova-cli costs
npx kova-cli sync
```

### From Source

Clone and build locally:

```bash
git clone https://github.com/klokare/kova-cli
cd kova-cli
npm install
npm run build
npm link
```

## Authenticate with the Dashboard

After installing, connect the CLI to your Kova cloud dashboard so usage syncs automatically.

**Step 1**: Create a free account at [kova.dev](https://kova.dev) if you have not already.

**Step 2**: Go to **Dashboard > Settings > API Keys** and create a CLI key.

**Step 3**: Run the login command and paste your key when prompted:

```bash
kova login
```

```
Enter your Kova API key: kova_xxxx...
Authenticated as: you@example.com (Free plan)
```

The key is stored locally in `~/.kova/credentials.json` and used automatically for all subsequent commands.

## Shell Completions

Enable tab completion for faster command entry:

```bash
# Bash
kova completions bash >> ~/.bashrc && source ~/.bashrc

# Zsh
kova completions zsh >> ~/.zshrc && source ~/.zshrc

# Fish
kova completions fish > ~/.config/fish/completions/kova.fish
```

## Verify Setup

Confirm the CLI is authenticated and can reach the API:

```bash
kova costs --since 7d
```

If you see `No usage records found`, authentication is working -- you just have not tracked any events yet. Follow the [Quick Start](/docs/getting-started/quickstart) to record your first usage event.
