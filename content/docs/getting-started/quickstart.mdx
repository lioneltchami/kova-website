# Quick Start

This guide walks through the full Kova FinOps workflow: authenticate, record your first usage event, sync it to the cloud, and view your cost breakdown in the dashboard.

## Step 1: Install and Authenticate

Install Kova and connect it to your dashboard:

```bash
npm install -g kova-cli
kova login
```

Paste your API key from **Dashboard > Settings > API Keys** when prompted.

## Step 2: Track a Usage Event

Record a cost event manually to test the setup. In practice, Kova collectors do this automatically in the background.

```bash
kova track --tool claude-code --model claude-sonnet-4-5 --input-tokens 12500 --output-tokens 3200 --cost 0.045
```

Expected output:

```
Recorded usage event
  Tool:    claude-code
  Model:   claude-sonnet-4-5
  Tokens:  12,500 in / 3,200 out
  Cost:    $0.045
  Stored:  ~/.kova/usage/2026-03/
```

## Step 3: View Your Costs

See a breakdown of what you have spent:

```bash
kova costs
```

Expected output:

```
AI Tool Costs -- March 2026
───────────────────────────────────────────────
Tool            Cost       Sessions   Tokens
claude-code     $0.045     1          15,700
───────────────────────────────────────────────
Total           $0.045     1          15,700

Period: 2026-03-01 to 2026-03-19
```

Filter by date range or tool:

```bash
kova costs --since 7d
kova costs --tool cursor --since 30d
kova costs --project my-app
```

## Step 4: Sync to the Cloud Dashboard

Push your local records to the cloud so they appear in your web dashboard:

```bash
kova sync
```

Expected output:

```
Syncing usage records...
  Uploaded: 1 record
  Accepted: 1
  Duplicates: 0
  Dashboard: https://kova.dev/dashboard
```

Visit [kova.dev/dashboard](https://kova.dev/dashboard) to see your costs with charts, filters, and team breakdowns.

## Step 5: Set a Budget

Create a monthly budget so you get an alert before overspending:

```bash
kova budget set --amount 100 --period monthly --warn-at 80
```

Expected output:

```
Budget created
  Amount:  $100.00 / month
  Alert:   Warn at 80% ($80.00)
  Synced to dashboard
```

When you reach 80% of your budget, Kova will warn you on the next `kova costs` or `kova sync` run.

## The Daily Workflow

Once collectors are configured for your tools (see [multi-tool tracking](/docs/guides/multi-tool-tracking)), the workflow is hands-off:

```bash
# Usage is recorded automatically by background collectors
# Sync at the end of the day (or run on a schedule)
kova sync

# Check your spend whenever you like
kova costs --since 7d

# Generate a report for the team
kova report --period monthly --format csv
```
