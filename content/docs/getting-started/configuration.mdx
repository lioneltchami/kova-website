# Configuration

Kova uses two configuration files:

- `~/.kova/config.yaml` -- global user config (collectors, auth, sync preferences)
- `~/.kova/budgets.yaml` -- budget definitions (managed by `kova budget` commands)

You rarely need to edit these files by hand. Use `kova config set` and `kova budget set` to make changes, or edit them in your text editor of choice.

## Global Config Overview

```yaml
version: 1

auth:
  api_key: kova_xxxx... # Set by kova login

collectors:
  claude_code:
    enabled: true
  cursor:
    enabled: false
    db_path: null # Auto-detected on macOS and Linux
  copilot:
    enabled: false
    monthly_cost: 19.00 # Your flat-rate Copilot subscription cost
  windsurf:
    enabled: false
    api_key: null
  devin:
    enabled: false
    api_key: null

sync:
  auto_sync: false # Run sync automatically after each kova track call
  sync_interval_hours: 24 # For scheduled/background sync (if auto_sync is true)
  upload_since_days: 7 # Default lookback window for kova sync

notifications:
  budget_warn:
    slack: null # Slack webhook URL for budget alerts
    discord: null # Discord webhook URL for budget alerts

display:
  currency: USD
  timezone: local # Timezone for date grouping (e.g. "America/New_York")
```

## Section Reference

### auth

Set automatically by `kova login`. Do not edit manually.

| Field     | Description                 |
| --------- | --------------------------- |
| `api_key` | Your Kova dashboard API key |

### collectors

Controls which AI tools Kova collects usage data from. See [Setting up multi-tool tracking](/docs/guides/multi-tool-tracking) for step-by-step instructions per tool.

| Field                  | Type    | Description                                    |
| ---------------------- | ------- | ---------------------------------------------- |
| `claude_code.enabled`  | boolean | Enable Claude Code tracking (default: true)    |
| `cursor.enabled`       | boolean | Enable Cursor tracking                         |
| `cursor.db_path`       | string  | Override auto-detected Cursor data path        |
| `copilot.enabled`      | boolean | Enable GitHub Copilot tracking                 |
| `copilot.monthly_cost` | number  | Your Copilot subscription cost per seat in USD |
| `windsurf.enabled`     | boolean | Enable Windsurf tracking                       |
| `windsurf.api_key`     | string  | Windsurf API key for usage polling             |
| `devin.enabled`        | boolean | Enable Devin tracking                          |
| `devin.api_key`        | string  | Devin read-only API key                        |

### sync

Controls how and when records are uploaded to the cloud.

| Field                 | Type    | Default | Description                                                   |
| --------------------- | ------- | ------- | ------------------------------------------------------------- |
| `auto_sync`           | boolean | `false` | Automatically sync after every tracked event                  |
| `sync_interval_hours` | number  | `24`    | Minimum hours between auto-syncs to avoid hitting rate limits |
| `upload_since_days`   | number  | `7`     | Default lookback window used by `kova sync` without `--since` |

### notifications

Webhook URLs for budget alert notifications. Alerts fire when your spend crosses a `warn_at` threshold defined in a budget.

```yaml
notifications:
  budget_warn:
    slack: "https://hooks.slack.com/services/TEAM/CHANNEL/TOKEN"
    discord: "https://discord.com/api/webhooks/ID/TOKEN"
```

### display

| Field      | Type   | Default | Description                                      |
| ---------- | ------ | ------- | ------------------------------------------------ |
| `currency` | string | `USD`   | Display currency (USD only in v0.3)              |
| `timezone` | string | local   | Timezone used for daily grouping in `kova costs` |

## Common Configuration Tasks

### Enable a Collector

```bash
kova config set collectors.cursor.enabled true
kova config set collectors.copilot.enabled true
kova config set collectors.copilot.monthly_cost 19.00
```

### Set a Slack Alert Channel

```bash
kova config set notifications.budget_warn.slack "https://hooks.slack.com/..."
```

### Change the Default Sync Window

```bash
kova config set sync.upload_since_days 30
```

### View Current Config

```bash
kova config
```
