# Cost Report Format

`kova report --format json` outputs a structured JSON document that summarizes AI tool spending for a period. Use this format to feed cost data into dashboards, finance systems, or custom scripts.

## Usage

```bash
kova report --format json
kova report --format json --period 2026-03
kova report --format json --output report.json
```

## Top-Level Fields

| Field           | Type   | Description                                                           |
| --------------- | ------ | --------------------------------------------------------------------- |
| `period`        | object | Date range covered by this report. See [period](#period).            |
| `total_cost_usd`| number | Total spend across all tools and models for the period.              |
| `total_sessions`| number | Total unique sessions across all tools.                              |
| `total_tokens`  | object | Aggregate token counts. See [total_tokens](#total_tokens).           |
| `by_tool`       | array  | Per-tool cost breakdown. See [by_tool](#by_tool).                    |
| `by_model`      | array  | Per-model cost breakdown. See [by_model](#by_model).                 |
| `by_project`    | array  | Per-project cost breakdown. See [by_project](#by_project).           |
| `generated_at`  | string | ISO 8601 timestamp when this report was generated.                   |

## period

| Field    | Type   | Description                     |
| -------- | ------ | ------------------------------- |
| `start`  | string | ISO 8601 start of the period.   |
| `end`    | string | ISO 8601 end of the period.     |
| `label`  | string | Human-readable label, e.g. `"March 2026"` or `"Last 30 Days"`. |

## total_tokens

| Field    | Type   | Description                          |
| -------- | ------ | ------------------------------------ |
| `input`  | number | Total input tokens across all tools. |
| `output` | number | Total output tokens across all tools.|

## by_tool

Array of objects, one per AI tool that had usage in the period. Sorted by `cost_usd` descending.

| Field      | Type   | Description                                          |
| ---------- | ------ | ---------------------------------------------------- |
| `tool`     | string | Tool identifier. See [Collectors Reference](/docs/reference/agent-types). |
| `cost_usd` | number | Total cost for this tool in the period.              |
| `sessions` | number | Unique session count for this tool.                  |
| `tokens`   | object | `{ input: number, output: number }` token totals.    |

## by_model

Array of objects, one per model that had usage in the period. Sorted by `cost_usd` descending.

| Field      | Type   | Description                                          |
| ---------- | ------ | ---------------------------------------------------- |
| `model`    | string | Normalized model name. See [Usage Data Format](/docs/reference/checkpoint-format). |
| `cost_usd` | number | Total cost for this model in the period.             |
| `requests` | number | Total request count (one per usage record).          |
| `tokens`   | object | `{ input: number, output: number }` token totals.    |

## by_project

Array of objects, one per project that had usage in the period. Projects with `null` project values are grouped under `"(unknown)"`. Sorted by `cost_usd` descending.

| Field      | Type   | Description                                          |
| ---------- | ------ | ---------------------------------------------------- |
| `project`  | string | Project name from the usage records.                 |
| `cost_usd` | number | Total cost for this project in the period.           |
| `sessions` | number | Unique session count for this project.               |
| `tools`    | array  | List of tool identifier strings used in this project.|

## Using the JSON Output

### Save to file

```bash
kova report --format json --period 2026-03 --output march-report.json
```

### Extract a single field with jq

```bash
kova report --format json | jq '.total_cost_usd'
```

### Get per-tool costs as a table

```bash
kova report --format json | jq -r '.by_tool[] | [.tool, .cost_usd] | @tsv'
```

### CI cost gate

```bash
COST=$(kova report --format json | jq '.total_cost_usd')
echo "Monthly spend: $COST"
```
