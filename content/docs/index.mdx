# Kova Documentation

Welcome to the Kova documentation. Kova is an AI Dev FinOps CLI that tracks cost and usage across every AI coding tool your team uses: Claude Code, Cursor, GitHub Copilot, Windsurf, and Devin.

Install once. Get a unified view of what your AI tools actually cost -- per developer, per project, per tool.

## Getting Started

- [Installation](/docs/getting-started/installation) - Install Kova CLI and start tracking in under 2 minutes
- [Quick Start](/docs/getting-started/quickstart) - Authenticate, run your first sync, and see your costs
- [Configuration](/docs/getting-started/configuration) - Set up budgets, alerts, and multi-tool collectors

## Commands

| Command            | Purpose                                                      |
| ------------------ | ------------------------------------------------------------ |
| `kova track`       | Record a usage event from any AI tool                        |
| `kova costs`       | Display cost breakdown by tool, model, project, or date      |
| `kova budget`      | Set and manage spending budgets with alert thresholds        |
| `kova sync`        | Push local usage records to the cloud dashboard              |
| `kova report`      | Generate a cost report for a period (CSV, JSON, or terminal) |
| `kova config`      | View and edit configuration                                  |
| `kova login`       | Authenticate the CLI with your Kova dashboard API key        |
| `kova completions` | Install shell tab completions                                |
