# Kova Documentation

Welcome to the Kova documentation. Kova is an AI Dev FinOps CLI that tracks cost and usage across every AI coding tool your team uses: Claude Code, Cursor, GitHub Copilot, Windsurf, Devin, Aider, Continue.dev, Cline, Amazon Q Developer, Bolt.new, and Lovable.

Install once. Get a unified view of what your AI tools actually cost -- per developer, per project, per tool.

- [Quick Start](/docs/getting-started/quickstart) - Authenticate, run your first sync, and see your costs
- [Configuration](/docs/getting-started/configuration) - Set up budgets, alerts, and multi-tool collectors

## Commands

| Command                  | Purpose                                                                  |
| ------------------------ | ------------------------------------------------------------------------ |
| `kova init`              | Interactive setup wizard -- detect tools and run your first cost scan    |
| `kova track`             | Scan and record AI tool usage data                                       |
| `kova costs`             | Display cost breakdown by tool, model, project, or date                  |
| `kova compare`           | Compare AI tool or model costs side-by-side with efficiency metrics      |
| `kova budget`            | Set and manage spending budgets with alert thresholds                    |
| `kova sync`              | Push local usage records to the cloud dashboard                          |
| `kova report`            | Generate a cost report for a period (CSV, JSON, or terminal)             |
| `kova audit`             | Export a full audit log of usage records in JSON or CSV                  |
| `kova ci-report`         | Generate a structured cost report for CI/CD pipelines                    |
| `kova tag`               | Map a project to a cost center for team budget tracking                  |
| `kova data export`       | Export your local usage database to a JSON file                          |
| `kova dashboard`         | Open the Kova web dashboard in your browser                              |
| `kova sso`               | Configure SSO authentication for enterprise environments                 |
| `kova policy`            | List, set, and enforce organization-level policies (Team/Enterprise)     |
| `kova config`            | View and edit configuration                                              |
| `kova login`             | Authenticate the CLI with your Kova dashboard API key                    |
| `kova logout`            | Remove stored credentials                                                |
| `kova account`           | View your Kova account and subscription details                          |
| `kova completions`       | Install shell tab completions                                            |

## Enterprise

Kova's Team and Enterprise plans add:

- **Cost centers** -- group project spending by team or product area using `kova tag`
- **RBAC** -- admin, member, and viewer roles with granular permissions
- **SSO** -- authenticate via Okta, Azure AD, Google Workspace, or any OIDC provider
- **Policy enforcement** -- require budgets and SSO across all developer machines with `kova policy enforce`
- **Org-wide reports** -- view spending across all team members in one dashboard

See the [Enterprise RBAC guide](/docs/guides/enterprise-rbac) and [kova.dev/pricing](https://kova.dev/pricing) for details.
